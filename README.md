# WDD430-Team04-Project: Handcrafted Haven
Team project repo for WDD430

## Folders
- `frontend/` Contains the Next.js and React modules
- `backend/` Contains the backend server.

#### frontend/
Next.js app runs on port `3000.` This frontend was created with the command `npx create-next-app@latest` using the defaults which include: `TypeScript`, `ESLint`, `No React Compiler`, `Tailwind CSS`, `No src/ directory`, and `App Router.`

#### backend/
Server uses port `8080.` The `backend/` uses javascript modules for the script type.

## How to
To get started with the `backend/` server:
  - `npm install` to install dependencies.
  - `npm run start` to start the server.

To get started with the `frontend/` app:
  - `npm install` to install dependencies.
  - `npm run build` to build Next.js app.
  - `npm run start` to start the app.

When running concurrently, start the backend and frontend on different terminals.
